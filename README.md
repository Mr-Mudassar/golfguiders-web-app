## 🎯 What It Is

GolfGuiders is the **digital clubhouse** for golfers — combining social features, GPS course navigation, AI assistance, e-commerce, and travel booking in a single platform.

This repository contains the **web application** (companion to the mobile apps).

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript |
| State Management | Redux Toolkit |
| Data Layer | GraphQL with Apollo Client |
| UI | Tailwind CSS, ShadCN, Radix UI |
| Real-time | WebSockets for live tournament updates |

## ✨ Key Features

- 🏌️ **Social feeds** — posts, stories, comments, reactions for the golfing community
- 🗺 **GPS course navigation** — interactive maps for thousands of golf courses
- 🤖 **Golfer GPT** — AI-powered golf assistant for tips, course advice, equipment recommendations
- 🏆 **Live tournament leaderboards** — real-time scoring with WebSocket updates
- 🛒 **Multi-vendor e-commerce marketplace** — equipment, apparel, accessories with shipping zones and wallet support
- 📺 **Video reels** — short-form golf content
- ✈️ **Trips travel-booking integration** — book golf trips and tournaments
- 🎓 **Golf coaching marketplace** — connect with certified instructors
- 👥 **Groups, events, and communities** — local golf clubs and meetups

## 🏗 What I Built

- Frontend implementation of major user-facing modules (feeds, marketplace, tournaments, profiles)
- GraphQL query/mutation/subscription integration with Apollo Client
- State management architecture using Redux Toolkit for complex shared state
- Reusable component library following ShadCN + Radix patterns
- Performance optimization for large data lists (virtualization, pagination, caching)
- Real-time leaderboard updates with WebSocket subscriptions
- Responsive design across desktop and mobile web

## 🎯 Scale & Impact

- Launched product with active users globally
- Supports thousands of golf courses with rich data
- Press coverage: featured in **Bellevue Today, March 2026**
