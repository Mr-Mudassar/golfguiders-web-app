This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses Tailwind CSS for its styling with Shadcn UI.

### Guidelines

Before pushing code run, the following commands to ensure code quality

```bash
# Runs ESLint to check for code quality and potential issues according to defined rules
yarn lint
# Checks for code formatting inconsistencies using Prettier without making changes
yarn format
# Automatically fixes code formatting issues based on Prettier rules
yarn format:fix
```
